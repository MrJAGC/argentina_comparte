Objetivo del proyecto:

El proyecto consiste en el diseño y desarrollo de una aplicación web denominada Portal
Latinoamérica Comparte – Argentina, orientada a replicar y adaptar el modelo de la organización
Colombia Comparte para el contexto argentino. Esta plataforma digital tiene como propósito
central conectar a personas, familias y empresas con programas de emprendimiento, liderazgo
y bienestar organizacional, promoviendo la reconstrucción económica y social de comunidades
vulnerables a través del empoderamiento productivo.

Descripción del sistema:

El sistema es una aplicación web desarrollada con el stack tecnológico Java 17 + Spring Boot +
PostgreSQL + Thymeleaf, bajo el patrón de arquitectura MVC de tres capas. Sus principales
componentes funcionales son:

• Módulo de Autenticación: login seguro con gestión de usuarios y roles (Administrador /
Usuario público).

• Módulo de Solicitudes de Contacto: formulario público para que interesados envíen
solicitudes, con panel de administración para consultar, responder y gestionar cada
solicitud.

• Módulo de Testimonios de Éxito: gestión de historias reales de personas beneficiadas
por los programas, con operaciones CRUD completas desde el panel administrativo.

• Módulo de Noticias: publicación y administración de artículos y novedades de la
organización, con visualización pública en el portal.

• Portal Público (Home): página de inicio con carrusel de banners, secciones informativas,
impacto en cifras y navegación general del sitio, adaptado al contexto argentino.
